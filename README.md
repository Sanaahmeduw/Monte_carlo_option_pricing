# Monte Carlo Option Pricing with Confidence Intervals

[![C++](https://img.shields.io/badge/C++-17-blue.svg)](https://isocpp.org/)
[![License](https://img.shields.io/badge/license-MIT-green.svg)](LICENSE)

## Overview

C++ implementation of Monte Carlo valuation for non-path dependent European options with statistical confidence interval estimation. Built as part of coursework for Masters in Quantitative Finance.

## Key Features

- **Monte Carlo simulation** for European option pricing
- **Confidence interval estimation** with customizable α level
- **Object-oriented design** with polymorphic `StatisticsMC` hierarchy
